## I-type instructions
I-type instructions have a 16 bit immediate field.

- BEQ
- BGEZ
- BGTZ
- BLEZ
- BLTZ
- BNE
- LB
- LBU
- LH
- LHU
- LW
- SB
- SLTI
- SLTIU
- SH
- SW

## R-type instructions
Main processor instructions which do not require a target address, imm value or branch displacement.

- MFHI
- MFLO
- MULT
- MULTU
- SLLV
- SLT
- SLTU
- SRAV
- SRLV
